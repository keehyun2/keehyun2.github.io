DB는 PostgreSQL 을 사용하는 것을 제안드립니다. 라이센스 문제가 전혀없습니다. MySQL은 오픈 소스 라이센스를 따르기는 하지만, 상업적으로 사용할 때는 상업용 라이센스를 구입해야만 합니다. 서버는 초기 비용이 저렴하고 사용량에 따라 추가 확장이 용이한 aws EC2 서비스를 사용하여 개발하는 것을 추천드립니다. 비용이 시간별로 사용량만큼 책정되고 사용하지 않을때는 비용이 부담되지 않습니다. 어플리케이션은 java springframework 을 사용하여 구현하겠습니다. 사용하여 개발된 내용을 사장님과 공유 하겠습니다. 기획서 보니까 깔끔하게 잘되어있어서 제안드립니다.
